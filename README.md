# 🌌 回憶轉運站 — 故事與夢想募資罐
「回憶轉運站」是一套將二手低效實體，轉化為人生成長虛擬燃料的**微型群眾募資平台**。本專案完全以 **React & Tailwind CSS** 打造，內建流暢的**公共前台**互動設計與深度的**管理後台狀態沙盒**，讓使用者、買家與管理員能以沈浸式的質感，輕鬆掌握與變更豐富的核心狀態。

---

## 🎨 設計理念與專案亮点

1. **感性懷舊與現代質感的和諧 Pairing**：
   - 以溫暖舒緩的**琥珀色 (Amber)**、沈穩的**大地石頭灰 (Stone)** 和象徵達標生機的**翡翠綠 (Emerald)** 作為主色調。
   - 字體選用極致清晰的 `Plus Jakarta Sans` 搭配代碼感、時間刻度感極強的 `JetBrains Mono` 作為統計數據襯底，營造具有「生活膠囊」與「時空檔案」的優雅格調。

2. **完整的前後台解耦與狀態互通 (Full-State Management)**：
   - **公共前台許願池 (Explore / Detail)**：供訪客瀏覽各項帶有時代印記的回憶信物（如：陶瓷紅豬撲滿、風扇高頻低吼的加密主機板、不合群的大一古典吉他）。訪客可以給予隨喜打賞（不拿實體物）或加碼競標。
   - **創作者主線進度條 (Dashboard / Portfolio)**：創作者能在這裡看見，所有釋出物件所累積的「打賞」與「搶標額」正即時化為柴火、推動著大目標（如日語輔導學費、沖繩郵輪票款）前進。
   - **系統狀態管理中心 (Admin System Controls)**：供內部或後台管理員調節進度、審核下線、更動實體狀態 (待審、運營、成交、封禁)、甚至是手動模擬外部買家出價金流。

3. **離線永續引擎 (Local Storage Synchronization)**：
   - 專案所有物件更迭、隨喜打賞記錄、競標歷史堆疊與狀態標籤都將**即時並自動同步於 `localStorage` 中**。即便重新整理瀏覽器，您的測試進度、自訂故事與圓夢感謝廣播日誌都將完好地被代代流傳！

---

## 🛠 系統狀態類型控制 & 模擬沙盒

平台完美支援並維護著四個核心維度狀態：
- **二手信物狀態** (`pending` 待審、`active` 開啟募得、`suspended` 被迫下線、`completed` 已成交)
- **實體追蹤狀態** (系統即時追蹤最高競標者 `highestBidder`，競標金額每被超越一次，舊出價即自動標記為 `refunded` 退還)
- **主線進度狀態** (只要收到打賞或更高級的出價，原擁有者的個人進度比例將**即時前進並觸發動態綠色渲染**；當達到 100% 時，立刻解鎖寄送圓夢電子明信片的限定回報畫面！)
- **即時日誌日誌** (任何更變在後台終端機都將以綠色代碼字元即時呈遞，保障測試的透明性與高度樂趣)

## 🚀 快速啟動
本應用程式已自動在 `Vite` 環境完成優雅構建，可以直接並行於 AI Studio 的 Sandbox preview 層：

```bash
# 執行開發伺服器
npm run dev

# 進行靜態編譯
npm run build
```

在右側的 Preview window 中即可直接暢玩！如果您想要重設一切測試進度、回復到最乾淨的初始四個感人故事狀態，只要切換到 **「系統：狀態管理中心」** 點選 **「重置預設資料與日誌」** 即可。
